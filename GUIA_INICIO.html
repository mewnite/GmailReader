📧 Gmail Watcher Pro

Monitor automático de correos en tiempo real

1 Instalar Dependencias

Primero, asegúrate de tener Python 3.7+ instalado. Luego ejecuta:

pip install plyer
¡Listo! Depende instalada

2 Crear App Password en Google

Este es el paso más importante. Necesitas una App Password especial.

Instrucciones (2 minutos):

  1. Abre: Google App Passwords
  2. Selecciona Correo en la primera opción
  3. Selecciona Windows en la segunda opción
  4. Google te generará una contraseña de 16 caracteres
  5. Cópiala SIN ESPACIOS

Ejemplo:

Generada: pqdt kgfp kcne zfrk Copia esto: pqdtkgfpkcnezfrk
Si no ves "App passwords", primero habilita "Verificación en 2 pasos" en Google Security

3 Ejecutar el Programa

Abre una terminal en la carpeta del programa y ejecuta:

python gmail_watcher_gui.py

Se abrirá una ventana con la interfaz gráfica.

4 Completar Configuración

En la ventana, completa estos campos:

Campo Ejemplo Notas
📧 Correo Gmail tu@gmail.com Tu correo de Gmail
🔑 App password pqdtkgfpkcnezfrk SIN ESPACIOS
👁️ Vigilar remitentes noreply@fing.edu.uy Separa varios con comas
⏱️ Intervalo 30 Segundos (mínimo 5)

5 ¡Iniciar Monitoreo!

Presiona el botón ▶️ Iniciar y listo.

Verás en la pestaña 📋 Alertas el estado actual:

  • 🟡 Conectando... - Se está inicializando
  • 🟢 Vigilando - ¡Funcionando!
  • 🔴 Detenida - No está activo
¡Ya está! El programa vigilará automáticamente

💡 Consejos Rápidos

  • Vigila múltiples remitentes con comas: email1@dom.com, email2@dom.com
  • O vigila por dominio: dom.com
  • Intervalo recomendado: 15-30 segundos
  • El programa sigue corriendo si lo minimizas
  • Las notificaciones aparecerán automáticamente

🐛 Solución de Problemas

  • No conecta: Verifica App Password SIN ESPACIOS
  • No detecta correos: Usa solo el dominio
  • Sin notificaciones: Presiona "🔔 Probar"
  • Se desconecta: Sube intervalo a 60s
  • Se congela: Reinicia el programa
